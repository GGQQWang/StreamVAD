## Loader 验证

已新增并上传两个 loader：

- `StreamVADStage1Dataset`
- `StreamVADStage2GateDataset`

默认只返回训练白名单字段。

Stage 1：

```text
video / clip_start / clip_end / target_text
```

Stage 2：

```text
video / chunk_start / chunk_end / gate_label
```

默认不会返回 `original_think`、`original_answer`、`original_start`、`original_end`、`trigger_reason` 等审计字段，避免训练时泄露。

服务器真实视频解码测试已通过：

```text
Stage1 frames shape: torch.Size([32, 720, 1280, 3])
Stage2 frames shape: torch.Size([8, 720, 1280, 3])
```

`torchvision/PyAV` 路径可能报 `Failed initializing scaling graph`，但 loader 已增加 `ffmpeg` fallback，最终可以正常解码。

## Adapter Smoke Test 命令

不解码视频，只检查字段：

```bash
python tools/smoke_test_streamvad_batches.py \
  --stage1-jsonl /data3/wgq/StreamVAD/data/streamvad_weak_supervision/streamvad_stage1_train.jsonl \
  --stage2-jsonl /data3/wgq/StreamVAD/data/streamvad_weak_supervision/streamvad_stage2_train.jsonl \
  --batch-size 2
```

真实解码视频片段：

